# Hetave Frontend

React + Vite frontend for Hetave Enterprises.

- Product catalogue with categories and detail pages
- Contact page with email integration
- Admin dashboard for managing products and categories

## Scripts

- `npm run dev` – start dev server
- `npm run build` – build for production
- `npm run preview` – preview production build

Set `VITE_API_URL` in `.env` or deployment environment to point to the backend API base URL.
