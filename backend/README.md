# Hetave Backend

Express + MongoDB backend for Hetave Enterprises.

- Uses MongoDB (Mongoose) for products, categories, orders, and contacts
- Stores images on Cloudinary
- Sends contact emails via Resend

## Scripts

- `npm run dev` – start in dev mode
- `npm start` – start in production mode

Configure environment in `.env` (Mongo URI, Cloudinary, Resend, JWT secret, FRONTEND_URL, etc.).
